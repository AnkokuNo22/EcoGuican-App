# 🏔️ EcoGüicán App
Plataforma digital para la gestión de ecoturismo y guías locales en **Güicán de la Sierra, Boyacá**.

### Funcionalidades:
- ✅ Registro de guías locales con persistencia de datos (AsyncStorage).
- ✅ Directorio dinámico de guías con fotos.
- ✅ Sistema de reservas vía WhatsApp con calendario integrado.
- ✅ Guía de preparación para turistas (Sistema de capas, cuidado del frailejón).

### Tech Stack:
- React Native + Expo
- Expo Router
- TypeScript
